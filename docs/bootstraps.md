# üßç Spacing Utilities
| Class  | Purpose                | Example |
| ------ | ---------------------- | ------- |
| `m-3`  | Margin on all sides    |         |
| `mt-2` | Margin-top             |         |
| `mb-4` | Margin-bottom          |         |
| `p-2`  | Padding on all sides   |         |
| `px-4` | Padding-left and right |         |
| `py-1` | Padding-top and bottom |         |

# üß≠ Navbar
| Class                      | Description                                    |
| -------------------------- | ---------------------------------------------- |
| `navbar`                   | Required on all navbars                        |
| `navbar-expand-lg`         | Navbar collapses on small screens (responsive) |
| `navbar-dark`              | Light text on dark background                  |
| `navbar-light`             | Dark text on light background                  |
| `navbar-brand`             | Logo/title area in the navbar                  |
| `navbar-nav`               | Container for nav links                        |
| `nav-item` / `nav-link`    | Individual items in the navbar                 |
